## 4. 범위

### 포함 (In Scope)
- `/coat next` 자동 실행 로직 (SKILL.md + commands/coat.md)
- server.js SSE 엔드포인트 추가 (`/api/events`)
- dashboard.html EventSource 전환
- wrap 로직에 README.md 버전 히스토리 업데이트 추가

### 제외 (Out of Scope)
- 대시보드 모바일 반응형
- WebSocket (SSE로 충분)
- 자동 버전 번호 파싱 (사용자가 버전명 직접 관리)
