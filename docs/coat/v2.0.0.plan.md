## 3. 성공 기준

- [ ] `npm run dashboard -- --projects /path/a,/path/b` 실행 시 두 프로젝트 모두 사이드바에 표시
- [ ] 사이드바 클릭으로 프로젝트 전환 — 모든 패널이 선택된 프로젝트 데이터로 교체
- [ ] 단일 프로젝트 모드(`--project`) 기존 동작 완전 유지
- [ ] VSCode 상태바에 🧥 `v2.0.0 · LOOP R3 | 기능97%` 형태로 표시
- [ ] `memory.json` 저장 즉시(1초 내) 상태바 텍스트 갱신
- [ ] 체크리스트 패널: 완료/미완료 항목 목록 + Match Rate
- [ ] `.coat/` 없는 폴더 열면 상태바 자동 숨김

## 4. 범위

### 포함 (In Scope)
- `server.js` — `--projects` 파싱, `/api/projects` 엔드포인트, 프로젝트별 state 반환
- `dashboard.html` — 사이드바 UI, 프로젝트 전환 로직
- VSCode Extension — `coat-statusbar` (신규 패키지)
  - `package.json` (contributes.statusBarItems)
  - `extension.ts` — memory.json watch + 상태바 갱신 + 체크리스트 패널

### 제외 (Out of Scope)
- 대시보드에서 프로젝트 추가/삭제 UI (CLI 인자로만)
- VSCode Extension 마켓플레이스 배포 (로컬 설치만)
- 상태바에서 직접 COAT 명령 실행
